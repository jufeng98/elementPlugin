参数 说明 类型 可选值 默认值
infinite-scroll-disabled 是否禁用(搭配 v-infinite-scroll 使用) boolean - false
infinite-scroll-delay 节流时延，单位为ms(搭配 v-infinite-scroll 使用) number - 200
infinite-scroll-distance 触发加载的距离阈值，单位为px(搭配 v-infinite-scroll 使用) number - 0
infinite-scroll-immediate 是否立即执行加载方法，以防初始状态下内容无法撑满容器。(搭配 v-infinite-scroll 使用) boolean - true
